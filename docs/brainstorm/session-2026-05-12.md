# Brainstorm Session: 2026-05-12

## Objective
- Trả lời câu hỏi về cấu trúc hệ thống (Web Server).
- Lên kế hoạch Việt hóa toàn bộ nội dung ghi chú (comments) trong mã nguồn Python và C++.

## Current Analysis
- **Hệ thống điều khiển hiện tại (Web Server)**: Qua phân tích `gui/main.py` và `firmware/src/main.cpp`, hệ thống **CHƯA CÓ** Web Server. Giao tiếp hiện tại đang thực hiện qua kết nối Serial (UART/USB) trực tiếp từ máy tính (chạy giao diện Python) xuống mạch ESP32. Mạch ESP32 chưa sử dụng thư viện `WiFi.h` hay `WebServer.h`.
- **Việt hóa Ghi chú code**: Mã nguồn đang sử dụng 100% comment tiếng Anh. Việc Việt hóa sẽ cần chỉnh sửa khoảng 16 tệp (bao gồm thư mục `gui/` và `firmware/`).

## Decisions
1. Hệ thống hiện chưa có Web Server (giao tiếp thuần qua Serial).
2. Quyết định (Override): Tiến hành Việt hóa toàn bộ comment trong code sang Tiếng Việt ngay lập tức.
3. Chấp thuận lộ trình tích hợp Web Server vào ESP32 ở Phase 2 (sử dụng WiFi và AsyncWebServer).
4. Sử dụng `/vp-auto` (hoặc chạy luồng tự động) để rà soát chất lượng sau khi Việt hóa và bắt đầu triển khai Phase 2.

## Phases
- **Phase 1: Việt hóa mã nguồn (Đang thực hiện)**
  - Quét toàn bộ file `.py`, `.cpp`, `.h`.
  - Dịch các inline comments, block comments sang tiếng Việt.
- **Phase 2: Tích hợp Web Server (Đã phê duyệt)**
  - Tích hợp module WiFi vào `firmware` (sử dụng `ESPAsyncWebServer`).
  - Thiết kế trang HTML tĩnh chứa các thanh trượt điều khiển Servo.
  - Viết API để frontend Web giao tiếp với backend ESP32 qua HTTP GET/POST hoặc WebSocket.
  - Đảm bảo có thể chạy song song điều khiển qua Web và Serial.
